Умный сервис прогноза погоды - Уровень со зведочкой
*видео работы представлено по ссылке https://drive.google.com/drive/folders/1tLeu2SzT1CatlOfNdMLM5ELFjBwB8zFg?usp=sharing
1) Проектирование серивиса. Для разработки будет использоваться язык Python с 
фреймворком Flask. Это будет сайт, размещенный на платформе Heroku.
2) На приветсвенной странице пользователю предложат ввести название города и
отметить в какой единице измерения показывать температуру (Кельвин/Цельсий).
После нажатия кнопки будет представлено две таблицы, основные параметры погоды и
предпочтительная одежда под эту погоду.
3)Процесс работы программы:
- Пользователь заходит на сайт и пишет название города(например Moscow), далее он
ставит/не ставит галочу напротив показывать в Цельсиях. После нажатия на кнопку
"find out the forecast". С помощью веб=формы данные отправляются в класс, 
предназначенный для формирования API-запроса, после получения ответа, он парсится 
специальным классом для выделения нужной информации. Записывается в базу данных.
Отправляется как параметр для функции рендеринга, также с помощью дополнительной
функции этого класса формируется предпочтительная одежда и также в качестве параметра
передается в функцию рендеринга. На выходе пользователь видит основные характеристики
погоды и рекомендации по одежде.
4) После заливания программы с гита достаточно запустить модуль runserver.py
для локальной работы.
